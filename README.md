# Vuln Scanner
This scrpit scanning target for XSS and open ports.

## Quick start
```bash
# Clone repo
git clone https://github.com/mental0-main/vulnscanner.git

# Move to direction
cd vulnscanner

# Run with python3
python3 vulnscanner.py
```

## Requirements
- Python3.8+
- Git
- Python-Requests
